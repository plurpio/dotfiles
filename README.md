### Installation
Execute the following commands to install my dotfiles on a **pre-installed** Arch installation. You will need to have **git** and **python3** (preinstalled) installed.
`git clone https://github.com/plurpio/dotfiles \
cd dotfiles/install/ \
python3 install.py`
